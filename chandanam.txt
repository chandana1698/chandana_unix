1.i-to insert
2.ESC-for command mode
3.x-to delete the unwanted character
4.u-to undo the last command and D to undo the whole line
5.CTRL-R to redo
6.A-to append text at the end
7.:wq-to save and exit
8.:q!-to trash all the changes
9.dw-move the cursor to the beggining of the word to delete that word
10.2w-to move the cursor two words forward
11.3e-to move the cursor to the end to the end of the third word forward
12.0(zero) to move to the start of the line
13.d2w-which deletes 2 words number can be changed for deleting the number of consecutive words like d3w
14.dd to delete the line and 2dd to delte to line number can be changed for deleting the n umber of consecutive words
15.p-puts the previously deleted text after the cursor (Type dd to delete the line and store it in a Vim register and p to put the line)
16.r-to replace the letter e.g press re to replace the letter with e
17.ce-to change until the end of a word
18.ce-deletes the word and places you in insert mode
19.G-to move you to the bottom of the file
20.gg-to move you to the start of the file.
21.%-to find a matching
22.:s/old/new/g to substitute 'new' for 'old' where g is globally
23./ backward search n to find the next occurence and N to search in opposite direction
24.?-forward search.
25.:|-to run shell commands like :|dir,:|ls
26.:w-TEST.Save the file
27.v-starts visual mode for selecting the lines and you can perform operationon that like d delete
28.:r-Filename will insert the content into the current file
29.R-to replace more than one character
30.y-operator to copy text using v visual mode and p to paste it.
31.yw-(copy)yanks one word
32.o-opens a line below the cursor and start insert mode
33-O-opens a line above the cursor 
34-a-inserts text after the cursor
35-A-inserts text after the end of the line
36.e-command moves to the end of a word
37.y-operator yanks(copies) text, p puts (pastes) it
38.R-enters Replace mode until<ESC>is pressed.
39.ctrl-w to jump from one window to another.
